# Contributing to TagWatchman

## フィードバック・バグ報告

バグ報告や機能要望は [Issues](../../issues) にてお願いします。

以下の情報を含めていただけると助かります：

- 発生している問題の概要
- 再現手順
- 期待する動作
- 実際の動作
- AWSリージョン・環境情報

## Pull Request について

現時点では Pull Request は受け付けていません。
